# Documentation

## Overview

The `docs/` directory contains the core documentation for the **Indonesian Language Quality Assurance Portfolio**.

These documents define the methodology, review process, quality framework, evaluation standards, and supporting concepts used throughout this repository.

Rather than focusing solely on proofreading examples, this documentation explains **how** language quality reviews are performed in a structured and repeatable manner.

---

# Purpose

The documentation serves several purposes:

* Establish a standardized review methodology.
* Define language quality standards.
* Ensure consistency across all case studies.
* Document review workflows.
* Provide guidance for proofreaders and reviewers.
* Support reproducible language quality evaluations.

---

# Scope

The documents in this directory apply to every folder within this repository, including:

* `cases/`
* `style-guide/`
* `templates/`
* `samples/`

All reviews and examples should follow the standards described here.
