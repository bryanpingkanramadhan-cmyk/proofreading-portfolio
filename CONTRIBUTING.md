# Contributing

## Reporting Issues

Each proofreading issue should include:

- Page number
- Section
- Original text
- Suggested revision
- Reason
- Severity
- Status

## Severity Levels

- Low
- Medium
- High
- Critical

## Categories

- Grammar
- Spelling
- Punctuation
- Typography
- Consistency
- Formatting
- References
- Data/Table

## Pull Requests

Please ensure every correction is supported by an explanation and follows the project's style guide.
