# Annual Report Proofreading

## Overview
This repository documents proofreading findings for an annual report in a structured and reproducible way.

## Scope
- Grammar
- Spelling
- Punctuation
- Consistency
- Typography
- Formatting
- Cross references
- Numbers and tables

## Repository Structure
```
docs/
reports/
before-after/
assets/
```

## Workflow
1. Identify issue.
2. Categorize it.
3. Propose correction.
4. Record rationale.
5. Verify resolution.
